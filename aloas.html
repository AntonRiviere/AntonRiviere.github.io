ALOAS - Site de gestion de tournois

Symfony Sulu Tailwind CSS

Contexte

Ce projet à été réalisé pour l'association d'un professeur de l'université, elle s'occupe d'handicapé et de personne à déficiance mentale. Le but de ce projet est de créer un site de gestion de tournoi pour plusieurs sports (pétanque,tennis,football) pour faciliter la mise en place de ces tournois lors de journée au sein de l'association. Le site devait permettre la gestion de tournoi mais aussi le fait que les handicapés puissent naviguer sur le site et voir où ils en sont dans le tournoi en fonction de leur logo d'équipe.

Mon travail

J'ai réalisé la création de tournoi de ce logiciel dans son ancienne version, il s'agissait d'un site en php natif.

  • Création de la base de données pour le site
  • Création de l'interface d'administration pour la gestion des tournois
  • Création de l'interface utilisateur pour la gestion des tournois
  • Création de l'interface utilisateur pour la gestion des équipes
